## Features

- Get  movies & TV shows
- Infinite scroll
- Fully responsive
- Watch movie & TV shows on torrent
- Player (Video.js)


## Future Features  
- Save movies & TV shows to tratk
 
 
## TMDb API connection setup

Add your TMDb API Key to .env.local/ REACT_APP_API_KEY


## Start

Install Dependencies
   
   npm install
    
Run App
   
   npm start

Build App
   
   npm build
